# ValidationRule 设计文档

## 类概述

ValidationRule 是数据验证规则的基类，用于定义自定义验证逻辑。

## 继承关系

```
无基类
```

## 核心职责

1. 定义验证接口
2. 执行验证逻辑
3. 返回验证结果

### 已实现功能 ✅

- ✅ 基本验证框架
- ✅ 验证结果类型

### 简单实现须扩充 ⚠️

### 未实现功能 ❌
