# RenderHost 设计文档

## 类概述

RenderHost 管理渲染宿主，连接UI系统和底层渲染后端。

## 继承关系

```
无基类
```

## 核心职责

1. 管理渲染窗口
2. 处理渲染循环
3. 管理渲染资源
4. 同步UI和渲染线程

## 实现状态

### 已实现功能 ✅

- ✅ 基本的窗口管理
- ✅ 简单的渲染循环
- ✅ 资源生命周期管理

### 简单实现须扩充 ⚠️

- ⚠️ 线程同步较简单
- ⚠️ 资源共享机制不完善

### 未实现功能 ❌

- ❌ 多窗口渲染
- ❌ 离屏渲染
- ❌ 渲染线程池
