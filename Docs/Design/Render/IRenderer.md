# IRenderer 设计文档

## 类概述

IRenderer 是渲染器的抽象接口，定义了渲染系统的标准API。

## 继承关系

```
无基类（纯接口）
```

## 核心职责

1. 定义渲染接口
2. 抽象渲染后端
3. 支持多种渲染实现

## 实现状态

### 已实现功能 ✅

- ✅ 核心渲染接口定义
- ✅ 几何体渲染接口
- ✅ 文本渲染接口

### 简单实现须扩充 ⚠️

- ⚠️ 接口粒度可优化

### 未实现功能 ❌
