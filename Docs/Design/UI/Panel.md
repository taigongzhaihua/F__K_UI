# Panel 设计文档

## 类概述

Panel 是 UI 模块的核心类，负责布局容器的基类。

## 继承关系

```
FrameworkElement → UIElement → Visual → DependencyObject
```

## 核心职责

1. 布局容器的基类
2. 子元素管理
3. 布局计算协调
4. 提供子元素集合
