# RenderCommand

## 概览

**目的**：表示单个渲染命令

**命名空间**：`fk::render`

**头文件**：`fk/render/RenderCommand.h`

## 描述

`RenderCommand` 封装单个绘图操作的数据。

## 命令类型

- DrawRectangle
- FillRectangle
- DrawEllipse
- FillEllipse
- DrawText
- PushTransform
- PopTransform

## 使用示例

内部由渲染系统使用。
