# PropertyStore

## 概览

**目的**：存储依赖属性值

**命名空间**：`fk::binding`

**头文件**：`fk/binding/PropertyStore.h`

## 描述

`PropertyStore` 是DependencyObject的内部存储，管理所有属性值。

## 功能

- 存储本地值
- 管理值优先级
- 处理继承值
- 缓存计算值

## 使用示例

内部由DependencyObject使用。
